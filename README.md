# Islamabad Traffic Police Scheduling System

## Overview

A console-based application developed in C++ to manage traffic police duty schedules. The project uses text files to store and retrieve scheduling information while demonstrating object-oriented programming and file handling concepts.


## Categories

- Administator/Owner
- Employeer/Outsider

## Features

- Add Police Schedule
- View All Schedules
- Search Schedule by Police ID
- Delete Schedule
- Duplicate ID Detection
- File-Based Data Storage

## Technologies Used

- C++
- File Handling
- Object-Oriented Programming
- Visual Studio Code
- MinGW (G++)

## Files

- main.cpp
- Scheduling_system.txt
- Check.txt

## How to Run

Compile:

```bash
g++ Project.cpp -o Project
```

Run:

```bash
.\Project
```

or in PowerShell:

```powershell
.\Project.exe
```
